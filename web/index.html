SimplyTranslate Web API

Translate

Path /api/translate

Arguments

  • engine

    • google

      Google Translate.

    • libre

      Libre Translate.

  • from

    Choose the From-Language (Source Language). It's based on the engine.

  • to

    Choose the To-Language (Target Language). It's based on the engine.

  • text

Ex: http://localhost:3000/api/translate/?engine=google&from=en&to=es&text=Hello

Get Languages

Path /api/get_languages

Arguments

  • engine

    • google

      Google Translate.

    • libre

      Libre Translate.

Ex: http://localhost:3000/api/get_languages/?engine=google
